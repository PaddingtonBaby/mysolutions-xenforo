# MySolutions

Плагин для XenForo 2.2, который делает счётчик решений в минипрофиле пользователя кликабельным.

## Ключевые преимущества

- Список тем, в которых автор оставил свое решение
- Теперь в закрытой теме решение изменить может только модератор или администратор
- Пагинация по списку
- Возможность выбора приватности отображения решений: все, только подписчики или никто, кроме пользователя.

## Установка

Скачать из релизов архив и установить через вкладку плагинов Xenforo.
